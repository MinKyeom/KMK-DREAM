==========================================================
        CURSOR 프로젝트 전체 스캔 및 분석 가이드
==========================================================

1. 핵심 기능: @Codebase
----------------------------------------------------------
- 사용법: 채팅창(Ctrl+L)이나 컴포저(Ctrl+I)에서 "@Codebase" 입력
- 역할: AI가 프로젝트 내의 모든 파일을 검색/분석하여 답변 생성
- 특징: 파일 간의 연결 관계(함수 호출, 의존성 등)를 파악함

2. 주요 활용 명령어 (예시)
----------------------------------------------------------
- "@Codebase 이 프로젝트의 전체 구조를 요약해줘"
- "@Codebase 특정 기능(예: 로그인)이 구현된 파일들을 모두 찾아줘"
- "@Codebase 프로젝트 전체에서 코드 스타일이 일관되지 않은 곳 찾아줘"
- "@Codebase 새로운 기능을 추가하려고 하는데, 어느 파일들을 수정해야 할까?"

3. 성능 최적화 (Indexing)
----------------------------------------------------------
- 인덱싱 확인: 설정(Ctrl+Shift+J) -> General -> Indexing
- 인덱싱이 'Ready' 상태여야 전체 스캔 답변이 정확해짐
- .cursorignore 파일: 스캔이 불필요한 폴더(node_modules, dist 등)를
  제외 설정하여 AI의 분석 속도와 정확도를 높임

4. 추천 모델 조합
----------------------------------------------------------
- 프로젝트 전체 분석 시: Claude Opus 4.6 (가장 꼼꼼하게 연결 고리 파악)
- 대규모 코드 수정 시: GPT 5.2 Codex (스캔 후 즉시 여러 파일 수정 실행)
